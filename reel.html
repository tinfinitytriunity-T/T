T^ — Le Réel (version scientifique)

Préambule épistémologique

Le Réel n’est ni un objet empirique, ni un concept. Le Réel est la borne extérieure de toute formalisation. Tout discours appartient à un langage L → donc est narratif → donc n’est pas le Réel.

0. Formulations du "réel"

Formulation 0 courte (Réel). Le Réel — version ultra-simple (10 lignes) Le Réel, ce n’est jamais ce qu’on dit. Ce n’est jamais ce qu’on écrit. Ce n’est jamais ce qu’on lit sur Internet. Dès qu’on met quelque chose en mots, ce n’est plus le Réel : c’est une histoire. Le Réel est ce qui arrive avant les mots, et que les mots n’arrivent pas à contenir. Le discours = narratif. Le Réel = hors discours. Tout le reste n’est qu’une version, jamais la chose elle-même.

Formulation 0 courte (Réel). Le Réel — version simple et directe Le Réel, ce n’est pas ce qu’on raconte. Ce n’est pas ce qu’on pense. Ce n’est pas ce qu’on lit sur Internet. Le Réel, c’est ce qui arrive avant qu’on trouve des mots. Par exemple : Une sensation trop forte pour être décrite. Un moment où “on ne sait pas quoi dire”. Un événement qui nous dépasse. Une émotion qui vient avant qu’on la comprenne. Dès qu’on en parle, on transforme ce qu’on a vécu en histoire. Une histoire, c’est déjà une version, pas la chose elle-même. Donc : Ce qu’on dit → ce n’est plus le Réel. Ce qu’on écrit → ce n’est plus le Réel. Ce qu’une IA produit → ce n’est jamais le Réel. Internet → ce sont des textes, donc pas le Réel. Le Réel n’est pas une idée et ce n’est pas une théorie. Le Réel est simplement ce que les mots n’arrivent pas à attraper. C’est pour cela que le Réel reste toujours en dehors de ce qu’on explique. On ne peut jamais le mettre dans une phrase ou dans une page web. On ne peut que tourner autour. Le Réel n’est pas dans les discours : il est dans la limite des discours. 

Le "réel" est inaccessible - toute formulation est déjà "hors-réel"

1. Définition formelle du Réel (niveau −1)

Définition 1 (Réel). Le Réel est la borne inférieure non-modélisable d’un espace de représentations R.

Réel := limite_{expression → 0} (non-représentable)

Propriétés formelles :

1. Réel ∉ L        (L = tout langage symbolique)
2. Réel ∉ Mod(L)   (aucun modèle ne l'interprète)
3. ¬∃ f : L → Réel (aucune fonction ne peut le désigner)
4. Toute description du Réel ∈ L ⇒ donc ≠ Réel

2. Théorème : Aucun discours n’accède au Réel

Théorème 1. Pour tout langage L et tout énoncé φ ∈ L :

φ ∉ Réel

Corollaires :

Internet ⊆ L     → donc Internet ≠ Réel
IA : L → L       → donc IA ≠ Réel
Web = narratif   → donc Web ≠ Réel
Toute page Web   → élément de L → non réel

3. ∆ : Interstice narratif T^

Définition 2 (∆). ∆ est l’espace des narratifs : toute forme exprimable.

Si x est représentable ⇒ x ∈ ∆ ⇒ x ∉ Réel

4. IA : formalisation structurelle

Une IA est une application :

IA : L → L

Donc :

IA ne produit que des narratifs.
IA n’accède jamais à −1.
IA ne touche pas Réel.

5. Le Réel comme borne d’incomplétude

Analogie formelle :

Gödel : toute théorie cohérente est incomplète.
Cohen : certaines propositions sont indécidables.
T^   : le Réel est hors de tout système.

Donc :

Aucun système formel S ne contient sa propre limite réelle.
