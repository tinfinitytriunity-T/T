Monde & Réel

1. Version ultra-simple

Quand on dit « le monde », ce n’est jamais le monde : c’est une façon d’en parler.

Quand on dit « le réel », ce n’est jamais le Réel : c’est encore une phrase.

Dès qu’on met quelque chose en mots, on est dans un narratif (∆). Ce narratif n’est pas ce qu’il prétend désigner.

Donc : • « le monde » = représentation. • « le réel » = représentation. • Le Réel, lui, reste hors de tout discours.

2. Version humaine

Le mot « monde » donne l’impression que nous parlons de quelque chose qui existerait tel quel, indépendamment de nous. Mais dès qu’on dit « monde », on utilise un langage : on produit donc une image, une version, une représentation.

Le même phénomène se produit quand on dit « le réel ». Le mot ne nous donne jamais accès à ce qu’il semble désigner. Il ne fait qu’ajouter un nouveau narratif de plus.

Le langage n’ouvre pas directement sur ce qui est. Le langage produit des formes, des concepts, des explications — toutes situées dans ∆.

Le Réel, lui, n’est pas une forme et n’entre jamais dans ces catégories. Il n’est jamais « ce dont on parle ». Il est ce qui rend toute parole insuffisante.

3. Définition scientifique : “monde” comme élément de L

Définition 1. Le mot « monde » appartient au langage (L). Toute occurrence du mot « monde » est donc un élément de L :

"monde" ∈ L

Conséquence directe :

Toute phrase contenant "monde" ∈ ∆
Toute phrase ∈ ∆ ≠ Réel (−1)

Le « monde » dont on parle n’est donc jamais le Réel. C’est une construction discursive.

4. Rappel : définition scientifique du Réel

Définition 2 (Réel). Le Réel est la borne inférieure non-représentable d’un système descriptif.

Réel := limite_{expression → 0} (non-formalisable)

Axiomes :

1. Réel ∉ L
2. Réel ∉ Mod(L)
3. Toute tentative de "dire" le Réel ∈ ∆
4. Donc : "le réel" (syntagme) ≠ Réel (−1)

5. Relation "monde" ↔ Réel

"monde" ∈ L
"le monde" ∈ L
"le réel" ∈ L

⇒ toutes ces expressions appartiennent à ∆
⇒ aucune n’est le Réel

La confusion disparaît dès que l’on distingue :

  • ce que l’on dit (narratif, ∆)
  • ce dont on croit parler (souvent fictif, construit)
  • ce qui n’entre jamais dans le langage (Réel, −1)

Le « monde » des discours ≠ un « monde » extérieur. Dans T^, nous ne décrivons que le premier.

6. Synthèse

Dire "monde" ⇒ langage ⇒ ∆ ⇒ représentation
Dire "réel"  ⇒ langage ⇒ ∆ ⇒ représentation

Rien dans ∆ n'est le Réel (−1)
Le Réel n'apparaît pas dans le langage
Toute expression est narratif

Cette page appartient elle-même entièrement à ∆. Elle ne dit rien du Réel : elle en indique seulement la limite.
